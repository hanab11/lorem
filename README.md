## vscode+git push报错 port 443: Timed out
>#### 解决办法：
>在HOST中删掉github代理地址，多次443可能是代理掉线需重连

## vscode+git push报错 SSL_read,,errno 10053
>#### 解决办法：
>其一git config http.sslVerify "false"
>其二git config --global --unset http.proxy
